What is this course?

Learn machine learning the practical way - both how things in ML works in math as well as how to apply them in real life.

What are the prerequisites?

Discrete Structures, Linear Algebra, Probability, and Intermediate Programming with Python.

Who are we?

This is a graduate level computer science course offered by Cornell Tech in New York City.
